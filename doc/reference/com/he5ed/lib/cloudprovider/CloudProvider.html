Class Overview

The main class that controls the entire operations of this library. This is a singleton class, only a single instance will be present across the entire lifecycle of the app that used this library. This class is also thread save thus can be called across different thread such as a Service.

To use this class for the first time its ACCOUNT_TYPE must be override to the host app's package name.

To interact with the cloud API backend service, build the respective cloud API class instance via this class's instance method buildApi(CloudAccount).

Developer should only call the account related operation via this class and should avoid direct interaction with the Android's AccountManager. All the account related operations are available via this class.
